**Focus Clothing** is a next-generation e-commerce platform designed for the Pi Network ecosystem. Founded in 2022, the brand bridges the gap between high-end fashion design and decentralized finance, allowing Pioneers to purchase premium apparel directly with Pi.

## 🚀 Vision
To empower local Zimbabwean designers and global creators by providing a secure, blockchain-verified marketplace that leverages the digital sovereignty of the Pi Network.

## ✨ Key Features
* **Web3 Payments:** Seamless integration with the Pi SDK for Testnet and Mainnet transactions.
* **Immersive Showcase:** High-fidelity visual catalogs optimized for mobile-first Pioneers.
* **Digital Sovereignty:** Secure authentication via the Pi Browser (KYC-ready).
* **Multi-Vendor Ready:** Scalable architecture to host a curated selection of fashion designers.

## 🛠️ Technical Stack
* **Frontend:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Blockchain Integration:** [Pi Network JavaScript SDK v2.0](https://minepi.com/)
* **Deployment:** Vercel / GitHub Actions
